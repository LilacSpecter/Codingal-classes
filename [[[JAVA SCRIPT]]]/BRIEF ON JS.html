Data types

data types are different kinds of datas.

JavaScript variables can have many data types like number, string, object, boolean.

number: any whole num, nums including decimals

strings: any set of characters grouped together inside '...' single quotes or "..." double quotes.

object: collection of data which are related.

boolean: boolean data type has only 'two possible values', either true or false

null: this data type represented by keyword null(without quotes).

underfined: this type represents the absence of value or not defined.
